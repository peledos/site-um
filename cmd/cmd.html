CMD

O que é?

Prompt de Comando é um interpretador de linha de comando no OS/2 e de sistemas baseados no Windows NT.

Exemplos de comandos

  1. dir - Lista os arquivos e pastas no diretório (pasta) atual;
  2. cd ou chdir - (Change Directory) - Muda o diretório atual;
  3. mkdir - (Make Directory) - criar um novo diretório;
  4. copy - Copia arquivos de um local para outro;
  5. del - Exclui um arquivo;
  6. ren - Renomeia um arquivo;
  7. notepad - abre o editor de texto do windows;
  8. write - abre o wordpad;
  9. type - Exibe o conteúdo de um arquivo de texto;
  10. cls - limpar a tela, removendo o histórico de comandos;
  11. echo - Exibe mensagens ou ativa/desativa o eco de comandos;
  12. type - Exibe o conteúdo de um arquivo de texto;
  13. ipconfig - Exibe informações de configuração de rede;
  14. ping - Testa a conectividade com um endereço IP;
  15. tracert - é usado para rastrear a rota que os pacotes de dados fazem da sua máquina até um destino especificado na rede;
  16. netstat - exibe uma lista das conexões de rede ativas;
  17. tree /f - Exibe a estrutura de diretórios da pasta atual;
  18. ver - Exibe a versão do Windows;
  19. attrib - Exibe ou modifica atributos de arquivos e diretórios;
  20. dir /s - Exibe uma lista completa de todos os arquivos no diretório atual e em todos os subdiretórios;
  21. dir /a /q - Exibe para mostrar todos os arquivos, incluindo os ocultos, além informações detalhadas do proprietário do arquivo;
  22. tasklist - Lista os processos em execução;
  23. taskkill - Permite encerrar um processo em execução;
  24. taskmgr - Abre o Gerenciador de Tarefas do Windows;
  25. assoc - usado para associar uma extensão de arquivo a um programa específico;
  26. shutdown - Permite desligar/reiniciar o computador à força;
  27. find - Permite pesquisar uma determinada sequência de caracteres em arquivos;
